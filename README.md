# GenAI Learning Repository

This repository contains my learning materials and experiments with Generative AI.

## Structure

- `l1_intorduction_to_genai/` - Introduction to GenAI concepts and basics

## Setup

1. Create a virtual environment:
   ```bash
   python -m venv venv
   ```

2. Activate the virtual environment:
   - Windows: `venv\Scripts\activate.bat`
   - PowerShell: `venv\Scripts\Activate.ps1`

3. Install dependencies:
   ```bash
   pip install -r l1_intorduction_to_genai/requirements.txt
   ```

## Requirements

- Python 3.14+
- See `requirements.txt` for package dependencies

## Environment Variables

Create a `.env` file in the project directories with your API keys and configuration:
```
# Add your environment variables here
```
